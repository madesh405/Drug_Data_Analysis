# Drug_Data_Analysis by E Madeshwaran

This project focuses on predicting **personalized drug treatment outcomes** across multiple medical conditions using machine learning techniques. The aim is to analyze how drugs, conditions, and reported side effects influence treatment effectiveness and to build predictive models that support data-driven healthcare analysis.

---

## Project Overview

- Predict treatment effectiveness based on drug and condition data  
- Analyze relationships between side effects and treatment outcomes  
- Compare drug performance across multiple medical conditions  
- Implement an end-to-end data science pipeline  

---

## Dataset

This project uses the **1000 Drugs and Side Effects** dataset from Kaggle.

- Platform: Kaggle  
- Dataset Author: Palak Jain  
- Data includes drug names, medical conditions, patient reviews, ratings, and reported side effects  

All credit for the dataset belongs to the original creator.  
The dataset is used strictly for educational and research purposes.
